Associate Data Practitioner Certification exam guide An Associate Data Practitioner secures and manages data on Google Cloud. This individual has experience using Google Cloud data services for tasks like data ingestion, transformation, pipeline management, analysis, machine learning, and visualization. Candidates have a basic understanding of cloud computing concepts like infrastructure as a service (IaaS), platform as a service (PaaS), and software as a service (SaaS).
